
# ComparteChela v1.0 

> Aplicación móvil para enviar cervezas a cualquier lugar y a cualquier persona al instante.

## Usage

### Phonegap Desktop / Phonegap CLI

Simply add the project to your projects directory or 
 run the command
 
 ```javascript
 	$ phonegap serve
```
